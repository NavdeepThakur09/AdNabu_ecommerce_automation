# AdNabu_ecommerce_automation
In this project, I automate the “Add to Cart” process on the AdNabu eCommerce website.

**Setup Instructions**
Method 1 (using CMD)
1. Install Python 3.13.9 and enable Add to Environment Variables during setup.
2. Verify installation: using command in cmd  
- python --version
3. Install Selenium using:
- pip install selenium
4. Go to project folder using cmd :
- cd AdNabu_task_2_Add_to_cart
5. Run the script using command:
- python AdNabu_Add_to_Cart_Task_2.py

Method 2 ( Install PyCharm)
1. Open project folder AdNabu_task_2_Add_to_cart in PyCharm
2. Set Python interpreter (virtual or system environment)
3. Install Selenium in PyCharm terminal: using pip install selenium
4. Run AdNabu_Add_to_Cart_Task_2.py
